# 安装方法
1. 克隆或下载
2. 压缩为 zip 压缩包之后在后台的主题设置直接上传即可使用。
3. 用Halo的程序拉取 [https://github.com/feiyangbeyond/halo-theme-sagiri.git](https://github.com/feiyangbeyond/halo-theme-sagiri.git)
> 感谢使用 halo-theme-sagiri 主题
